# 项目管理页面 UI 优化

参照 DaVinci Resolve 的设计风格进行了全面优化。

## 主要改进

### 1. 布局优化
- **网格布局**：从 2 列布局改为 2-4 列响应式网格（根据屏幕尺寸）
  - 移动端：2 列
  - 平板：3 列
  - 桌面：4 列
- **更大的对话框**：从 `max-w-5xl` 增加到 `max-w-7xl`，提供更宽敞的视觉空间
- **卡片间距**：从 `gap-4` 增加到 `gap-6`，视觉层次更清晰

### 2. 项目卡片重新设计

#### 原设计
```
┌─────────────────────────┐
│ 项目名称          [状态] │
│ 项目描述               │
├─────────────────────────┤
│ 📅 2026-02-10           │
│ 📖 50 章  🎬 5 集      │
│ [查看] [删除]          │
└─────────────────────────┘
```

#### 新设计（参照 DaVinci Resolve）
```
┌─────────────────────────┐
│                         │
│    [预览区域]           │
│     (图标背景)      [状态]│
│   [⚙️菜单]              │
│                         │
├─────────────────────────┤
│ 项目名称                │
│ 项目描述                │
│ 📖 50 章  🎬 5 集       │
│ 📅 2026-02-10           │
└─────────────────────────┘
```

### 3. 视觉增强

#### 预览区域
- **aspect-video** (16:9 比例) 预览区
- 渐变背景：`from-primary/10 via-accent/10 to-muted`
- 大型文件夹图标作为占位符
- 悬停时显示操作菜单

#### 交互效果
- **悬停缩放**：`hover:scale-[1.02]`
- **阴影增强**：`hover:shadow-xl`
- **边框高亮**：`hover:border-primary/50`
- **颜色过渡**：项目名称悬停时变为主题色

#### 操作菜单
- 右上角 `⋮` 图标菜单（悬停时显示）
- 下拉菜单形式，操作更简洁
- 替代了原来底部的两个按钮

### 4. 创建项目卡片

#### 特殊设计
- **虚线边框**：`border-2 border-dashed`
- **大型 + 图标**：`size-16` 的加号
- **悬停效果**：图标和文字颜色变为主题色
- **占位卡片**：与其他项目卡片并列显示

### 5. 创建表单优化
- 从独立按钮改为内嵌卡片形式
- 更大的间距和视觉层次
- 自动聚焦到项目名称输入框
- 更明显的边框样式（虚线 + 背景色）

### 6. 空状态优化
- 更大的空状态图标：`size-20`
- 更清晰的文字层次
- 大号创建按钮：`size="lg"`
- 居中显示，视觉更突出

## 用户体验提升

1. **更专业的视觉**：参照行业标准软件（DaVinci Resolve）的设计语言
2. **更清晰的层次**：预览区 + 信息区分离，一目了然
3. **更流畅的交互**：悬停效果、缩放动画、颜色过渡
4. **更高效的操作**：悬停菜单减少视觉噪音，操作更快捷
5. **更好的响应式**：4 档网格布局适配不同屏幕

## 对比总结

| 项目 | 原设计 | 新设计 |
|------|--------|--------|
| 布局 | 2 列固定 | 2-4 列响应式 |
| 卡片高度 | 自适应 | 统一 aspect-video |
| 操作方式 | 底部按钮 | 悬停菜单 |
| 视觉重点 | 文字信息 | 预览区域 |
| 创建按钮 | 独立按钮 | 卡片形式 |
| 动画效果 | 基础阴影 | 缩放+阴影+颜色 |
